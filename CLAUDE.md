# Bedbeards

Brand: Barbershop  
Stack: Stack 1 — React + Vite + TypeScript + Tailwind CSS + React Router v6

## Rules
- pnpm only
- .tsx only
- All text/data in src/data/siteContent.ts
- All colors as Tailwind tokens in tailwind.config.ts
- src/components/layout/ → Navbar, Footer, ScrollToTop
- src/components/ui/ → Button, Divider, FadeInSection, PageHero, SectionHeader
- src/components/home/ → homepage sections
- src/pages/ → flat .tsx files only (no folder/index pattern)
- src/styles/index.css → CSS entry point
- App.tsx → routing only
